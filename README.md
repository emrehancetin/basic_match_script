Random Matching Tool (GUI)

1. Put your names in a text file (one name per line). Example: names.txt
2. Run the app:
   python match_gui.py

Notes:

- You can select any .txt file via Browse.
- Mode:
  - auto: pairs if even, assignments if odd
  - pairs: requires even number of names
  - assignments: works for any count >= 2
- Save Results button exports output as a .txt file.
